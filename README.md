# Expense Tracker

**Автор:** [Бондаренко Никита]
**Дата создания:** Апрель 2026

## Описание

Приложение для отслеживания личных расходов с графическим интерфейсом. Позволяет:
* Добавлять расходы с указанием суммы, категории и даты.
* Фильтровать записи по категории и периоду.
* Подсчитывать общую сумму расходов за выбранный период.
* Сохранять и загружать данные в формате JSON.

## Как использовать

1. Запустите приложение: `python main.py`.
2. Заполните поля:
   * **Сумма** — введите положительное число.
   * **Категория** — выберите из списка или введите свою.
   * **Дата** — укажите в формате ГГГГ-ММ-ДД.
3. Нажмите **«Добавить расход»** для сохранения записи.
4. Используйте фильтры для отбора данных:
   * Выберите категорию из выпадающего списка.
   * Укажите период в полях «С даты» и «По дату».
5. Нажмите **«Применить фильтр»** для обновления таблицы.
6. Нажмите **«Подсчитать сумму за период»** для расчёта общей суммы.

## Примеры использования

**Тест 1: Добавление расхода**
1. Введите: Сумма = 500, Категория = Еда, Дата = 2024-01-15.
2. Нажмите «Добавить расход».
3. Убедитесь, что запись появилась в таблице.

## Особенности реализации

* **Сохранение данных:** все расходы сохраняются в файле `expenses.json` в формате JSON между запусками приложения.
* **Валидация:** проверка:
   * заполнения всех полей;
   * положительной суммы расхода;
   * корректного формата даты (`ГГГГ ММ ДД`).
* **Гибкость категорий:** пользователь может выбирать из предустановленных или вводить свои категории.
* **Фильтрация:** возможность комбинировать фильтры по категории и дате.
* **Подсчёт суммы:** автоматический расчёт общей суммы расходов за выбранный период с учётом активных фильтров.
* **Интуитивный интерфейс:** логичное расположение элементов, понятные подсказки и сообщения об ошибках.
* **Автоматическое присвоение ID:** каждой записи присваивается уникальный номер.
* **Очистка полей:** после успешного добавления расхода поля ввода очищаются для удобства следующего ввода.
